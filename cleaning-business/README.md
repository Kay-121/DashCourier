# Pro Clean Crew - Private Website

A private, locally-hosted website for Pro Clean Crew cleaning services.

## How to View

Simply double-click `index.html` to open the website in your browser. No internet connection required after first load (though CDN fonts/icons will cache).

## File Structure

- `index.html` - Main webpage
- `style.css` - Styling
- `script.js` - Interactive features

## Features

- Responsive design for all devices
- Service listings with pricing
- Quote request form
- Contact information
- Smooth animations

## Privacy

This website is designed for local use only. No hosting configuration is included.
